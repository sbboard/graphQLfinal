# Starting Database
```
npm run dockerStart
npm run deploy
```
After deploying the GraphQL Playground can be accessed on http://192.168.99.100:4499/
# Seed
```
npm run load
```
Runs a script that loads data from [KuroganeHammer's Smash Bros 4 API](https://github.com/Frannsoft/FrannHammer/wiki)
# Reset Database
```
npm run reset
```
